# CSI（Control Sequence Introducer）
### ascii样式控制字符速查

1. 克隆仓库

~~~bash
git clone https://github.com/AkashiNeko/CSI.git
~~~

2. 运行

~~~bash
cd CSI/
./csi-list
~~~
